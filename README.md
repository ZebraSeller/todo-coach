# todo-coach
Todo Coach is a Todo List App powered by an AI Assistant (Gemini 2.5 Flash Lite) using Google Cloud's Vertex AI. 

## Tech Stacks
- The frontend is built with React and Tailwind, it is responsive and up to modern standards with clean state management. It delivers a intuitive and sleek UI for a smooth user experience.
- The backend is built with Express.js, and is deployed on Google's Cloud Run as a containerized serverless app for efficient and reliable performance.
- The app manages user authentication and task data storage using Google Cloud Firestore, leveraging a NoSQL structure for fast, flexible data retrieval.
- Integrated Vertex AI and the Gemini model to provide context-aware task summarization and advice.

## Commands
To Run this app, first navigate to the frontend folder, then use the following command:
```
npm run dev
```
If errors are present, try running this first:
```
npm install --legacy-peer-deps
```
